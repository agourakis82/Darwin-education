  # Darwin Logo Prompt Pack v1 (30 Prompts)

  Use this pack to generate high-end logo candidates with consistent quality.

  ## Global generation settings

  - Style target: premium, minimal, geometric, calm, clinical-tech.
  - Output: transparent background, clean edges, centered composition.
  - Avoid: mockups, 3D chrome, glass text, lens flares, cliche medical icons.
  - Keep visual language consistent with Darwin palette:
    - Graphite `#0B0F14`
    - Slate `#1B2430`
    - Emerald `#00B894`
    - White `#F7F9FC`

  ## Prompt rules

  - For symbol-only prompts: do not include text.
  - For lockup prompts: text must be exactly `Darwin Education`.
  - Always request vector-clean styling in the prompt.

  ## Recommended generation workflow

  1. Generate 4 variants per prompt.
  2. Filter by 24px readability and silhouette first.
  3. Keep top 12 candidates.
  4. Score with `scripts/brand/score_logo_candidates.ts`.
  5. Build final family from top 2 candidates.
